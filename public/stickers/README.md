# Stickers para Fotobook

Esta carpeta contiene los stickers disponibles para usar en el editor de páginas.

## Cómo agregar tus propios stickers

1. Coloca archivos de imagen (PNG, SVG, JPG) en esta carpeta
2. Usa nombres descriptivos (ej: `avion.png`, `corazon.svg`, `estrella.png`)
3. Tamaño recomendado: 200x200 px a 500x500 px
4. Formato preferido: PNG con fondo transparente o SVG

## Stickers incluidos

Los stickers por defecto están en formato de emojis para viajes y celebraciones:
- `plane.png` - Avión (viajes)
- `camera.png` - Cámara (fotos)
- `heart.png` - Corazón (amor/amistad)
- `star.png` - Estrella (destacados)
- `balloon.png` - Globo (celebraciones)
- `palm-tree.png` - Palmera (playas/vacaciones)
- `graduation.png` - Birrete (graduación/prom)
- `dance.png` - Baile (fiesta/prom)

## Notas

- Los stickers se pueden mover, rotar y redimensionar en el editor
- No se pueden colocar en los bordes de la página
- Se guardan junto con las fotos en el caché del navegador
